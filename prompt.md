# Prompt

## Create New Project

### Step 1

* Buatkan dokument SRS dalam bentuk file markdown berdasarkan dokument pdf tersebut
* Bahasa Pemograman : php terbaru
* Framework : Laravel Starter Kit Terbaru ( Inertia + Vue JS + Shadcnui)
* Database : mariadb
* Container : docker
* Gunakan the best practice dari laravel
* Buatkan arsitektur project the best practice dari web development, buat agar mudah dikembangkan di masa yang akan datang
* Gunakan clean code
* Implementasikan unit testing
* Setup docker mode dev dan mode prod

### Step 2

* Buatkan check list task implementasi bertahap berdasarkan dokument SRS tersebut

### Step 3

* Buatkan juga AGENT.md atau CLAUDE.md untuk menyimpan context dari project tersebut sabagai base informasi ke AI Agent.
